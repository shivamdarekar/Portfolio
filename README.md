# Shivam Darekar's Portfolio Website

A modern, responsive portfolio website built with Next.js 15, React 19, TypeScript, and Tailwind CSS v4, showcasing my journey as a Computer Science Engineering student, full-stack developer, and tech enthusiast.

🏆 Status: COMPLETE & PRODUCTION-READY

Fully optimized for performance, SEO, accessibility, and real-world usage.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server (with Turbopack)
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Type checking
npm run type-check

# Code formatting
npm run format
```

**Local Development:** http://localhost:3000
